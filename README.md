# CampusConnect LPU

CampusConnect LPU is a networking platform built for LPU students to discover and connect with each other through LinkedIn.

## Features

- Google Sign-In
- Student Profiles
- LinkedIn Profile Integration
- Active Members Directory
- Search & Filters
- Responsive Design

## Tech Stack

- React
- Vite
- Tailwind CSS
- Supabase
- Google Authentication
